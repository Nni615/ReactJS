2)What is a marquee in HTMl? explain with two examples?

->Marquee means we can scroll the part of text or the whole document in different directions like up up, down, left and right.

->It consists attributes like behaviour, directions, ScrollAmount and ScrollDelay along with values of it.

->Behaviour attribute consists three values i.e. "Alternate","Scroll", "Slide"(By default slide will be the value of it).

->Directions attribute consits four values i.e "Left", "Right","Up" and "Down".

->ScrollAmount attribute is use to increase the speed of the text, while ScrollDelay will decrease the speed of the text.

->For ex.

<marquee behaviour="scroll" direction="right"> HTML TAGS & ELEMENTS HTML TAGS & ELEMENTS
