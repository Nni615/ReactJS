9)Define z-index? create examples as look like examples using z-index.

->It is the property of css that controls the order of overlapping elements on a page.

->An element with a higher z-index value will appear in front of an element with the lower z-index value.

->If you have two elements with a z-index 0 and 1, the element with a z-index of 1 will be the front of an element with a z-index of 0.
