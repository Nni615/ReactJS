->Html contains three types of list i.e. Ordered list(ol),Unordered list(ul) and Defination list(dl).

->Ordered list means all the content will be in an sequence manner.

->Unordered list means all the content will be in an unsequence manneror we can say not in an order way.

->In an Unordered list content will be in an bullets form.

->Defination list also known as an Description list ,it means content contain details like title and data of it.
